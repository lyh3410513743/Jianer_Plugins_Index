# GuardianWords - 智能敏感词检测插件

### GuardianWords 是一款基于 AC 自动机算法开发的高效群聊敏感词检测插件，支持 API 自动检测敏感词，支持多维度敏感词管理、违规行为管控、自动化处罚等功能，适配 Hyper 机器人框架，守护群聊语言环境。

## 🌟 核心功能

1. #### 高效敏感词检测：基于 AC 自动机实现多敏感词快速匹配，支持大小写不敏感检测
2. #### 精细化权限控制：区分管理员/普通用户权限，白名单用户豁免检测
3. #### 群维度开关：支持不同群聊独立开启/关闭敏感词检测功能
4. #### 违规行为管控：
   
   - 违规消息自动撤回
   - 多维度违规统计（时间窗口/违规次数）
   - 达到阈值自动禁言
   - 违规记录持久化存储
5. #### 灵活配置管理：支持自定义违规统计窗口、最大违规次数、禁言时长等参数
6. #### 完整的管理指令：敏感词增删、白名单管理、违规记录查询/重置等

## 🚀 快速部署

1. #### 安装依赖

  在命令行执行：`pip install -r requirements.txt`

## 📖 使用指南

#### 管理员指令（前缀为配置的 reminder）

`{reminder} 敏感词检测               查看当前群检测状态及配置
{reminder} 敏感词检测 开启          开启当前群敏感词检测
{reminder} 敏感词检测 关闭          关闭当前群敏感词检测
{reminder} 敏感词检测 状态          查看当前群检测状态
{reminder} 敏感词检测 添加敏感词 [词]  添加敏感词到本地库
{reminder} 敏感词检测 删除敏感词 [词]  从本地库删除敏感词
{reminder} 敏感词检测 添加白名单 [QQ号] 将用户加入白名单（豁免检测）
{reminder} 敏感词检测 删除白名单 [QQ号] 将用户移出白名单
{reminder} 敏感词检测 重置用户违规 [QQ号] 清空指定用户的违规记录
{reminder} 敏感词检测 查看违规记录 [QQ号] 查看指定用户的违规详情
{reminder} 敏感词检测 设置 窗口时间 [秒数] 设置违规统计时间窗口
{reminder} 敏感词检测 设置 最大违规 [次数] 设置触发禁言的最大违规次数
`{reminder} 敏感词检测 设置 禁言时长 [秒数] 设置违规禁言时长

### 示例指令

`/敏感词检测 开启
/敏感词检测 添加敏感词 广告
/敏感词检测 设置 禁言时长 300
/敏感词检测 查看违规记录 123456789`
