# BilibiliUserSearch - 哔哩哔哩用户查询插件

## ✨ 功能特点

- 🔍 通过UID查询B站用户信息
- 📊 显示用户基本信息：昵称、性别、等级、签名
- 🧮 计算粉丝关注比并给出评价
- 🎭 自动识别大会员身份
- ⏳ 内置5秒用户冷却时间
- 📸 支持显示用户头像
- 🎯 中文提示配合表情符号，阅读体验更佳
  

## 🚀 使用方法

在群聊中发送以下格式的消息：

```
{reminder}查B站用户 [UID]
```



例如：

```
/查B站用户 401742377
```
