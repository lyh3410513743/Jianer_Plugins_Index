# B站投稿助手 (Bilibili Archives Assistant)

一个用于查询B站UP主投稿视频的机器人插件，支持关键词搜索和分页浏览。

## ✨ 功能特性

- 🔍 **快速查询**：通过B站用户mid快速查询投稿视频列表
- 🔎 **关键词搜索**：可在UP主的投稿中搜索特定关键词
- 📄 **分页浏览**：支持分页查看，每页显示5条视频
- 📊 **信息详细**：显示视频标题、播放量、时长、发布时间和短链接
- ⚡ **响应迅速**：异步处理，实时反馈查询状态
- 🛡️ **错误处理**：完善的错误处理和用户提示

## 📖 使用说明

### 基本命令格式

## `查投稿 [mid] [关键词] [页码]`

### 参数说明
- **mid**：B站用户的数字ID（必填）
- **关键词**：可选，搜索UP主投稿中的特定内容
- **页码**：可选，默认为1，指定要查看的页面

### 使用示例
1. **查询用户投稿**：`查投稿 1969160969`
2. **查询特定页面**：`查投稿 1969160969 2`
3. **关键词搜索**：`查投稿 1969160969 原神`
4. **搜索并分页**：`查投稿 1969160969 原神 2`

### 查询结果展示
查询结果会以清晰格式展示：
- 视频标题（自动截断过长内容）
- 播放量（自动转换为"万"单位）
- 视频时长
- 发布时间
- B站短链接
- 分页导航信息
