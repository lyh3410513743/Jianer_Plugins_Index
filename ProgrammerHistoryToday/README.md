# 程序员历史上的今天插件

这是一个用于QQ机器人的插件，当用户在群聊中发送"程序员历史上的今天"时，会返回当天在计算机科学和编程历史上发生的重要事件。

## 功能特点

- 实时查询程序员历史上的今天事件
- 显示事件的年份、标题、描述、分类和重要性等级
- 按重要性显示星级评价（⭐）
- 支持事件标签显示
- 包含美观的消息格式和友好提示
- 完善的错误处理机制
