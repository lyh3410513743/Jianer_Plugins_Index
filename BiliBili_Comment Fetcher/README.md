# BiliBili Comment Fetcher - B站评论查询插件



## ✨ 功能特性

- 🔍 查询B站视频评论
- 📄 支持分页显示（每页5条）
- 📱 显示用户等级、点赞数、时间、IP属地
- 🖼️ 识别图片评论
- ⚡ 异步请求，响应快速
- ❌ 完善的错误处理

## 📦 安装依赖

1. 确保已安装Python 3.7+
2. 安装所需依赖：
```bash
pip install aiohttp
```

## 🎮 使用方法

### 基本命令格式

```
/b站评论 [视频ID] [页码]
```



### 示例

```
/b站评论 115852649174965      # 查看第1页
/b站评论 115852649174965 2    # 查看第2页
```
