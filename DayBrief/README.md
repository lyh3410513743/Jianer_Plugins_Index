# 插件名称：**DayBrief**

## 功能描述：

本插件用于从指定API获取当日新闻摘要图片，并在群聊中发送该图片。当用户在群聊中发送触发关键词“日新闻图”时，插件会自动请求新闻图片API，并将获取到的图片发送至当前群聊。

## 使用方法：

在群聊中发送以下关键词即可触发：

- 日新闻图

返回内容：
- 成功时：发送一张包含当日新闻摘要的图片（格式通常为JPG）
- 失败时：根据错误类型返回相应的错误提示信息

## 错误提示包括：

1. API返回非图片数据
2. 服务器内部错误（500）
3. 新闻源获取失败（502）
4. 请求超时
5. 网络请求异常
6. 插件执行意外错误

## 注意事项：

- 由于新闻生成可能需要一定时间，若请求超时，建议稍后重试
- 图片为临时生成，发送后会自动清理临时文件
- 请确保网络畅通，以便正常访问外部API
